To launch the project:

Clone the repository (git clone https://github.com/AndreiDudenkov/project_for_iOS).
Open the project in your IDE.
Install dependencies (npm install). Node version v18.15.0, npm version 9.5.0.
Start the project (npm run start).
To view the layouts, change the URL in the browser to:
http://localhost:3000/index.iPhone_8.html?lang=en for iPhone SE, 8+
http://localhost:3000/index.iPhone_13.html?lang=en for iPhone 11 Pro, 14, 14 Plus, 14 Pro Max.
To change language settings, modify the browser language or URL (e.g., http://localhost:3000/index.iPhone_13.html?lang=es).
The project is designed for English, with automatic language detection based on browser settings and URL for language switching. Gulp is used as a build tool.
